# RAG-from-scratch
rag_from_scratch_1 demonstrates the core steps of building a Retrieval-Augmented Generation (RAG) pipeline — including indexing, retrieval, and generation — based on the tutorial from github https://github.com/langchain-ai/rag-from-scratch/tree/main?tab=readme-ov-file.
.
Unlike the original version that uses the OpenAI API, this implementation integrates the free Gemini API instead.
It has also been updated to work with the latest LangChain version, reflecting the recent module and import changes.
